# YouTube AI Automation & Video Generator

Open-source YouTube content automation tool with AI video generation.

**Inspired by Seedance 2 style** — automatic full video creation from text.

## Features (in development)
- AI script generation
- Text-to-Video & Image-to-Video
- Automatic voiceover
- YouTube upload automation (title, description, tags, thumbnail)
- Batch video production
- Multi-language support

## Tech Stack
- Python
- AI APIs (Claude, Grok, etc.)
- MoviePy / FFmpeg
- Selenium / YouTube API

## Goal
Help content creators scale their YouTube channels with minimal effort.

**Status:** Actively developing. Main maintainer: @o3-noone
